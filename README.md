# Family Planner Telegram Bot

Telegram-бот для семейного планирования: календарь, задачи, покупки, обязанности, меню, финансы, заметки, напоминания и AI-помощник через OpenAI API.

## Возможности MVP

- Быстрое меню Telegram вместо запоминания команд.
- "Сегодня" и "Неделя" показывают семейные события, задачи, покупки, вишлисты и напоминания.
- Кнопки "Событие", "Задача", "Покупки", "Вишлист", "Напоминание" открывают пошаговые конструкторы.
- AI может разобрать обычную фразу и подготовить запись на подтверждение.
- `/digest` показывает семейную сводку на день, а `/digest week` - на ближайшие 7 дней.
- `/export` отправляет семейный календарь файлом `.ics` для импорта в календарные приложения.
- "Напоминание" создает персональное напоминание.
- Если кто-то добавляет событие, бронь, задачу, покупку, маркетплейс, вишлист, финансовую запись, меню, заметку или напоминание, бот отправляет уведомление в семейный чат.
- Фоновая проверка напоминаний отправляет уведомления в Telegram.

## Установка

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
```

Заполните `.env`:

```env
TELEGRAM_BOT_TOKEN=токен_от_BotFather
OPENAI_API_KEY=ключ_OpenAI
OPENAI_MODEL=gpt-5.2
DAILY_DIGEST_TIME=
DAILY_DIGEST_CHAT_IDS=
ALLOWED_CHAT_IDS=
ALLOWED_USER_IDS=
```

`OPENAI_API_KEY` можно оставить пустым, тогда планер будет работать, а кнопка "AI" сообщит, что AI не настроен.

## Запуск

```powershell
python -m family_planner_bot
```

## Быстрое меню

После `/start` бот показывает кнопки:

```text
☀️ Сегодня | 📅 Неделя
👤 Мои задачи | ✅ Все задачи
🗓️ Событие | ✅ Задача | 🛒 Покупки
🎁 Вишлист | 🔔 Напоминание
```

Нажмите кнопку и заполните конструктор по шагам. Для необязательных полей отправьте `-`.
После последнего шага бот покажет итог и кнопки `Сохранить`, `Изменить`, `Отмена`.
Запись попадет в планер только после подтверждения.

Если указан `OPENAI_API_KEY`, можно писать запись обычной фразой без выбора кнопки:

```text
завтра в 18:00 у Маши стоматолог, напомнить за час
до пятницы Папе оплатить ЖКУ, повторять каждый месяц
добавь в вишлист Маме массажер для спины
добавь на маркетплейс WB кроссовки ребенку, 3200 рублей, ссылка позже
купить молоко 2 литра
напомни завтра в 9 маме взять документы
```

AI распознает событие, задачу, покупку, маркетплейс, вишлист или напоминание и покажет тот же экран проверки. Команда `/add` делает то же самое в запасном режиме.

В списках `Сегодня` и `Неделя` каждая запись показывается отдельной карточкой с кнопками:

```text
Выполнено | Изменить
Перенести | Удалить
```

`Удалить` дополнительно спрашивает подтверждение. `Изменить` меняет название записи, `Перенести` меняет дату.

`Мои задачи` показывает только задачи, назначенные на текущего Telegram-пользователя. Для этого пользователь должен один раз зарегистрироваться через `/join Имя`. `Все задачи` показывает все открытые семейные задачи.

Для событий и задач конструктор предлагает напоминание заранее: за 10 минут, за 1 час, за 1 день или не напоминать. Если рассчитанное время уже прошло, дополнительное напоминание не создается.

Примеры ответов в конструкторах:

```text
Событие:
1. завтра 18:30
2. Мама
3. врач
4. health
5. -
6. за 1 час

Задача:
1. до пятницы
2. Папа
3. оплатить ЖКУ
4. finance
5. monthly
6. за 1 день

Покупки:
1. молоко
2. 2 л

Вишлист:
1. Мама
2. массажер для спины
3. день рождения
4. ссылка или комментарий

Напоминание:
1. завтра 09:00
2. Мама
3. взять документы в школу
```

Формат с `|` остался только в запасных командах, чтобы быстро вводить записи одной строкой.

## Семейный дайджест

```text
/digest
/digest week
```

Если `OPENAI_API_KEY` настроен, бот превратит ближайшие записи и напоминания в короткий план: главное, риски, что подготовить, покупки и напоминания. Если AI не настроен, команда вернет обычную сгруппированную сводку без обращения к OpenAI.

Чтобы бот сам отправлял дайджест каждое утро, задайте время в `.env`:

```env
DAILY_DIGEST_TIME=08:30
DAILY_DIGEST_CHAT_IDS=-1001234567890
```

Время берется в локальной таймзоне сервера. Если `DAILY_DIGEST_CHAT_IDS` пустой, бот отправит дайджест во все чаты, которые уже есть в базе через записи, напоминания или членов семьи.

## Экспорт календаря

```text
/export
/export week
/export 90
```

Команда отправляет `.ics` файл с ближайшими записями, у которых есть дата: события, брони, задачи, меню, финансы и напоминания. По умолчанию экспортируются 30 дней, `week` - 7 дней, число задает период от 1 до 180 дней.

## Повторяющиеся события и задачи

Для `/event` и `/task` можно добавить повтор в конце:

```text
/event вторник 17:00 | Ребенок | футбол | секции | weekly
/task завтра 20:00 | Папа | вынести мусор | дом | daily
```

Поддерживаются `daily`, `weekly`, `monthly`.

## Где хранятся данные

По умолчанию база SQLite создается в `family_planner.sqlite3` рядом с проектом. Путь можно поменять через `DATABASE_PATH`.

## Доступ

Если `ALLOWED_CHAT_IDS` и `ALLOWED_USER_IDS` пустые, бот доступен всем, кто может ему написать.

Чтобы закрыть доступ:

1. Напишите боту `/id` из нужного семейного чата или личного диалога.
2. Скопируйте `chat_id` или `user_id` в `.env`.
3. Перезапустите бота.

Пример:

```env
ALLOWED_CHAT_IDS=-1001234567890
ALLOWED_USER_IDS=111111111,222222222
```

Команда `/id` остается доступной всем, чтобы новый член семьи мог прислать свой ID владельцу бота.

## Члены семьи и упоминания

Чтобы бот предлагал выбрать человека кнопкой в конструкторах задач, событий, вишлистов и напоминаний, каждый член семьи должен зарегистрироваться в нужном чате:

```text
/join Мама
```

Если имя не указать, бот возьмет имя из Telegram:

```text
/join
```

После этого на шаге "кому" бот покажет inline-кнопки с членами семьи. Если у человека есть Telegram user_id, уведомления будут содержать кликабельное упоминание.

## OpenAI

AI-помощник использует OpenAI Responses API: бот передает краткий снимок семейного планера для ответов на вопросы и отдельный структурированный запрос для разбора свободной фразы в запись планера. Запись сохраняется только после подтверждения пользователем.
